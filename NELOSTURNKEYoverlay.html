1. System Architecture & Overview

Non-Euclidean Layered Operating System (NELOS) Framework

NELOS (Non-Euclidean Layered Operating System) is an open-source high-throughput state mapping framework designed to stream data vectors directly between storage media and high-performance processing nodes (GPUs / Accelerators).

By utilizing dynamic bitwise bitmap indexing and sparse vector state mapping, the NELOS Overlay FS bridges standard host platforms—including Chromebooks running Linux development environments—with high-density compute pipelines, bypassing traditional OS file-system locks.

Direct Access (DMA)

Routes 32GB USB flash media streams straight into GPU VRAM buffers using memory-mapped zero-copy operations.

4. Chromebook Linux Environment Guide

Step-by-Step Crostini Container Provisioning

  1. Enable Chromebook Developer Environment:

    Navigate to ChromeOS Settings > Advanced > Developers > Turn on "Linux development environment". Allocate at least 20 GB of container storage.

  2. Mount External 400 GB Drive (Optional):

    Plug in your external drive, right-click it in the Files app, and select "Share with Linux".

  3. Update Container & Install Build Dependencies:
    sudo apt update && sudo apt upgrade -y
    sudo apt install python3-pip python3-venv build-essential parted udev -y
  4. Create Isolated Python Virtual Environment:
    python3 -m venv nelos_env
    source nelos_env/bin/activate
    pip install numpy amd-quark

5. OverlayFS & Möbius Drive Mechanism

Non-Euclidean Bitwise Mapping Theory

Sparse Bitmap Indexing

Instead of writing traditional block sectors to the 32GB USB flash media, the NELOS filesystem generates sparse bitmap indices. Only active non-zero tensor points are written, allowing dense mathematical matrices to occupy minimal physical disk footprint.

Möbius Continuous Folding

Data streaming follows a cyclic boundary condition. When continuous telemetry streams exceed allocated physical RAM windows, state indices wrap continuously into the virtual mapping tables without triggering system boundary interrupts.
